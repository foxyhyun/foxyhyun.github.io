알게된 내용들 : 
- ill-posed : Image Prior, 이미지가 어떻게 생겼어야 하는지에 대한 사전 지식
- local connectivity : 각 뉴런이 인접한 pixel들만 본다
- translation equivariance : 위치에 무관하게 패턴 인식
- Receptive Field 개념에서 이미지의 멀리 있는 내용도 같이 보려면 Layer를 깊게 쌓아야 함
- Convolution Filter는 static weights at inference로 입력에 유연하지 않다
- Self Attention은 전체 이미지의 모든 픽셀 쌍간 유사도를 구하므로, Receptive Field가 전체 이미지고 입력에 동적임
- vanilla multi-head SA는 공간 차원이 아닌 K와 Q로 입력된 입력 feature channel간 cross covariance 계산함
- local context mixing : 각 feature의 pixel을 계산할 때, 주변 픽셀들의 정보를 섞어 사용
- pixel unshuffle : 정보 손실 없이 공간 해상도를 줄이되 채널 수를 늘림
- pixel shuffle : max pooling, binary interpolation 대신 사용
- 연산량 때문에 SA를 Image Restoration에 사용을 잘 안함
- depth wise convolution : 각 픽셀의 feature에 주변 픽셀이 봄
- bias free convolutional layers : bias 항이 있으면 다양한 noise level에 성능이 떨어짐. 입력의 절대값이 아닌, 상대적 패턴에 집중할 수 있음
- GELU : 0에 가까우면 해당 위치의 F 억제, 1에 가까우면 통과
- progressive learning : train, test의 이미지 해상도가 다른 문제 때문에 inference에서 성능이 떨어질 수 있음. 따라서 학습 때도 본 적 없는 size도 가능케 세팅하는 방법
- curriculum learning : 쉬운거부터 어려운순으로 학습

중요하다고 생각하는 문장들 : 
- we propose an efficient Transformer model by making several key designs in the building blocks (multi-head attention and feed-forward network) such that it can capture
long-range pixel interactions, while still remaining applicable to large images
- Although SA is highly effective in capturing long-range pixel interactions, its complexity grows quadratically with the spatial
resolution, therefore making it infeasible to apply to highresolution images 
- In this paper, we propose an efficient Transformer for image restoration that is capable of modeling global connectivity and is still applicable to large images. Specifically, we introduce a multi-Dconv head ‘transposed’ attention (MDTA) block (Sec. 3.1) in place of vanilla multi-head SA [77], that has linear complexity. It applies SA across feature dimension rather than the spatial dimension, i.e., instead of explicitly modeling pairwise pixel interactions,
MDTA computes cross-covariance across feature channels to obtain attention map from the (key and query projected)
input features. An important feature of our MDTA block is the local context mixing before feature covariance computation. This is achieved via pixel-wise aggregation of crosschannel context using 1×1 convolution and channel-wise
aggregation of local context using efficient depth-wise convolutions. This strategy provides two key advantages. First, it emphasizes on the spatially local context and brings in the
complimentary strength of convolution operation within our pipeline. Second, it ensures that the contextualized global relationships between pixels are implicitly modeled while
computing covariance-based attention maps
- we reformulate the first linear transformation layer of the regular FN [77] with a gating mechanism [16] to improve the information flow through the network.

- The key ingredient is to apply SA across channels rather than the spatial dimension, i.e., to compute cross-covariance across channels
