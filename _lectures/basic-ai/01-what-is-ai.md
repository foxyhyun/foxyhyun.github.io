## AI란 무엇인가?

**인공지능(Artificial Intelligence, AI)**은 인간의 지능적 행동을 컴퓨터가 흉내내도록 만드는 기술의 총칭입니다.

"지능적 행동"에는 다음이 포함됩니다.

- 언어 이해 및 생성
- 이미지 인식
- 문제 풀기 / 추론
- 게임 플레이
- 의사 결정

---

## AI vs. ML vs. DL

세 개념은 포함 관계입니다.

```
┌─────────────────────────────────────┐
│            AI (인공지능)              │
│   ┌───────────────────────────┐     │
│   │    Machine Learning       │     │
│   │   ┌─────────────────┐     │     │
│   │   │  Deep Learning  │     │     │
│   │   └─────────────────┘     │     │
│   └───────────────────────────┘     │
└─────────────────────────────────────┘
```

| 용어 | 핵심 아이디어 |
|---|---|
| **AI** | 사람처럼 행동하는 시스템 전반 |
| **Machine Learning** | 데이터로부터 규칙을 **스스로 학습** |
| **Deep Learning** | 인공 신경망(Neural Network)을 깊게 쌓아 학습 |

---

## 전통적 프로그래밍 vs. ML

### 전통 방식
```
입력 데이터 + 규칙(사람이 작성) → 출력
```

### 머신러닝 방식
```
입력 데이터 + 원하는 출력 → 규칙(모델)을 자동으로 학습
```

> 예: "스팸 메일 필터"를 만든다면
> - **전통 방식**: 사람이 "광고", "무료" 같은 단어 목록을 직접 입력
> - **ML 방식**: 수천 개의 스팸/정상 메일을 보여주면 컴퓨터가 패턴을 스스로 발견

---

## AI의 세 가지 학습 방식

### 1. 지도 학습 (Supervised Learning)
- **레이블이 있는** 데이터로 학습
- 예: (이미지, 정답 클래스) 쌍을 이용한 분류

### 2. 비지도 학습 (Unsupervised Learning)
- 레이블 **없이** 데이터의 구조/패턴을 발견
- 예: 고객 군집화, 이상 탐지

### 3. 강화 학습 (Reinforcement Learning)
- 환경과 **상호작용**하며 보상 최대화
- 예: AlphaGo, 게임 플레이 AI
